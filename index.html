Project/Thesis Details

Title - Detection of Ocular Diseases using Neural Network.

In this project, our Neural Network detected Ocular diseases such as Pathological myopia, Age related macular degenration(AMD), Diabetic retinopathy(DR), Glaucoma, Catract with 87.25% accuracy. We used Convolutional Neural Network(CNN) and Naive bayes algorithms. CNN was used for image processing, initial-prediction and Naive bayes was used for final-prediction based on the inputs given by CNN. Fundus images gathered from sources like kaggle.com, google.com , were used for trainig and testing. Our project was coded in Java for implementing Neural Network algorithms, HTML, CSS, Java-Script for UI design, Hadoop for backend and database. We have published paper on this project on International Journal of Innovative Research in Engineering & Multidisciplinary Physical Sciences(IJIRMPS).
